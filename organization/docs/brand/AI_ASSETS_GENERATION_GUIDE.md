# Broxiva AI Assets Generation Guide

## Overview

This guide outlines the AI-generated creative assets required for the Broxiva e-commerce platform. These assets will enhance brand perception, user engagement, and conversion rates.

---

## 1. AI Avatars

### Brand Assistant Avatar

**Purpose:** Customer support chat, onboarding guides, help sections

**Specifications:**
- Style: Modern, professional, approachable
- Gender: Neutral/diverse options
- Attire: Business casual with navy/gold accents
- Expression: Friendly, confident, helpful
- Formats: PNG (transparent), SVG, MP4 (animated)
- Sizes: 64x64, 128x128, 256x256, 512x512

**Prompt Template:**
```
A professional, modern AI assistant avatar for a premium e-commerce platform.
The avatar should appear friendly and trustworthy, wearing business casual attire
with deep navy (#1a365d) and gold (#c9a227) accent colors. Clean, minimalist style
with soft lighting. High quality, suitable for web use.
```

### Shopping Guide Avatar

**Purpose:** Product recommendations, style advice, shopping assistance

**Specifications:**
- Style: Fashion-forward, sophisticated
- Diverse representation options
- Premium, editorial quality
- Multiple poses: standing, gesturing, presenting

## 3. AI-Generated Videos

### Brand Introduction Video (30 seconds)

**Purpose:** Homepage hero, about page, social media

**Storyboard:**
1. (0-5s) Broxiva logo animation with gold shimmer
2. (5-15s) Product showcase montage, premium transitions
3. (15-25s) Global reach visualization, world connectivity
4. (25-30s) CTA with tagline "Global Premium E-Commerce, Delivered"

**Specifications:**
- Resolution: 1920x1080, 4K optional
- Frame Rate: 30fps
- Audio: Elegant, modern soundtrack
- Style: Cinematic, premium

### Product Explainer Videos (10-15 seconds)

**Purpose:** Product pages, social ads, email marketing

**Format:**
- Square (1:1) for social
- Portrait (9:16) for stories
- Landscape (16:9) for web

**Elements:**
- Product 360 rotation
- Feature callouts with gold highlights
- Smooth transitions
- Subtle motion graphics

### Short-Form Ads (10 seconds)

**Purpose:** Social media advertising, retargeting

**Structure:**
- Hook (0-2s): Eye-catching visual
- Product (2-7s): Key benefits
- CTA (7-10s): Clear action

## 6. Technical Specifications

### Image Formats

| Use Case | Format | Quality |
|----------|--------|---------|
| Web Display | WebP | 80-90% |
| Fallback | JPEG | 85% |
| Transparency | PNG | Lossless |
| Vector | SVG | - |
| Icons | SVG/PNG | Lossless |

### Video Formats

| Platform | Format | Codec | Bitrate |
|----------|--------|-------|---------|
| Web | MP4 | H.264 | 8-12 Mbps |
| Mobile | MP4 | H.265 | 4-6 Mbps |
| Social | MP4 | H.264 | 6-10 Mbps |

### Responsive Sizes

**Hero Images:**
- Mobile: 640x360
- Tablet: 1024x576
- Desktop: 1920x1080
- Retina: 3840x2160

**Product Images:**
- Thumbnail: 150x150
- Grid: 400x400
- Detail: 800x800
- Zoom: 1600x1600
