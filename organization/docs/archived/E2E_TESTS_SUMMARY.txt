HOW TO RUN TESTS:
================================================================================

API Tests:
  cd organization/apps/api
  pnpm test:e2e

Frontend Tests:
  cd organization/apps/web
  pnpm test:e2e

Run Specific Test File:
  API: pnpm test:e2e -- test/shopping.e2e-spec.ts
  Web: npx playwright test e2e/shopping.spec.ts

  npx playwright test --ui
