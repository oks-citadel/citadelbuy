## AI-Generated Product Images

### Product Photography Style

**Lighting:**
- Soft, diffused key light
- Subtle rim lighting for dimension
- Clean, shadow-controlled setup
- Premium studio lighting simulation

**Backgrounds:**
| Type | Usage | Color |
|------|-------|-------|
| Pure White | Main product shots | #FFFFFF |

**Angles Required:**
1. Front view (hero shot)
2. 3/4 angle view
3. Side profile

5. Detail close-ups

**Electronics:**
```
Resolution: 2000x2000px minimum

Additional: Reflections, screen-on versions
```


**Home & Living:**
```
Resolution: 2000x2000px
Format: JPEG with lifestyle context
Additional: Room scene renders
```

**Beauty/Cosmetics:**
```
Resolution: 1500x1500px minimum
Format: PNG with reflections
Additional: Texture close-ups, swatches
```

## AI-Generated Video Specifications

### 1. Product Videos

**Specifications:**
```
Resolution: 4K (3840x2160) master, 1080p deliverable
Frame Rate: 30fps (standard), 60fps (premium)
Duration: 15-30 seconds
Format: MP4 (H.264)
Aspect Ratios: 16:9 (desktop), 9:16 (mobile), 1:1 (social)
```

**Content Structure:**
1. Opening brand moment (2-3 seconds)
2. Product reveal (3-5 seconds)
3. Feature highlights (10-15 seconds)
4. CTA/brand close (3-5 seconds)

**Style Guidelines:**
- Smooth camera movements (dolly, orbit)
- Clean transitions (dissolve, morph)
- Subtle motion graphics
- Premium color grading

### 2. Hero Banner Videos

**Specifications:**
```
Resolution: 1920x600 (desktop), 750x1000 (mobile)
Duration: 8-15 seconds (seamless loop)
Format: WebM (primary), MP4 (fallback)
File Size: Under 5MB for web
```

**Content Requirements:**
- Seamless loop capability
- Minimal movement (battery friendly)
- Text-safe zones maintained
- Brand elements subtly incorporated

### 3. Social Media Videos

**Instagram/TikTok:**
```
Resolution: 1080x1920 (9:16)
Duration: 15-60 seconds
Format: MP4
Frame Rate: 30fps
```

**YouTube:**
```
Resolution: 3840x2160 (4K)
Duration: 30 seconds - 5 minutes
Format: MP4
Frame Rate: 30fps
```

**Twitter/X:**
```
Resolution: 1920x1080 (16:9)
Duration: 15-60 seconds
Format: MP4
File Size: Under 512MB
```

### 4. AI Avatar Videos

**Talking Head Videos:**
```
Resolution: 1080x1080 (square) or 1080x1920 (vertical)
Duration: 30-120 seconds
Lip Sync: AI-generated, natural movement
Background: Virtual studio or transparent
```

**Use Cases:**
- Product announcements
- Customer service responses
- Personalized recommendations
- Brand messages

```
Size: 1200x800 (landscape), 800x1200 (portrait)

Usage: Landing pages, marketing
Style: Detailed, aspirational
```

## Quality Assurance Checklist

### Avatar Quality Standards

- [ ] Consistent facial features across all poses
- [ ] Natural skin textures and tones
- [ ] Realistic eye reflections and highlights
- [ ] Proper clothing fabric rendering
- [ ] Clean transparent backgrounds
- [ ] No visual artifacts or distortions
- [ ] Consistent lighting across set
- [ ] Brand color accuracy

### Product Image Standards

- [ ] Accurate product representation
- [ ] Clean edges and cutouts
- [ ] Consistent white balance
- [ ] Sharp focus throughout
- [ ] No compression artifacts
- [ ] Color accuracy maintained
- [ ] Proper file naming convention
- [ ] All required angles provided

### Video Quality Standards

- [ ] Stable frame rate throughout
- [ ] Clean audio (if applicable)
- [ ] Smooth transitions
- [ ] Proper color grading
- [ ] Correct aspect ratios
- [ ] Optimized file sizes
- [ ] Brand guidelines followed
- [ ] Call-to-action visible

## File Naming Convention

### Avatars
```
brx-avatar-[name]-[pose]-[expression]-[size].png
Example: brx-avatar-aria-001-happy-1024.png
```

### Product Images
```
brx-product-[sku]-[angle]-[variant].png
Example: brx-product-SKU123-front-white.png
```

### Videos
```
brx-video-[type]-[name]-[resolution]-[version].mp4
Example: brx-video-product-headphones-1080p-v1.mp4
```

### Illustrations
```
brx-illustration-[category]-[name]-[size].svg
Example: brx-illustration-icon-cart-48.svg
```
