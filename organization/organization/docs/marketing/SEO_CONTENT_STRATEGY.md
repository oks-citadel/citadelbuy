## 1. SEO Strategy Overview

### 1.1 SEO Objectives

**Primary Goals**:
- Achieve 500,000 organic monthly visits by Q4 2027
- Rank in top 3 for 50+ high-intent B2B keywords by Q2 2027
- Generate 5,000+ organic SQLs per month by Q4 2027
- Establish Broxiva as the thought leader in Africa-global B2B trade

**Target Keywords**: 200+ primary keywords, 1,000+ long-tail keywords

### 1.2 SEO Philosophy

**Approach**: **Topic Clusters + Pillar Pages**
- Create comprehensive pillar pages targeting broad topics
- Support with detailed cluster content targeting long-tail keywords
- Internal linking strategy to establish topical authority

**Content-First SEO**:
- Prioritize user intent over keyword density
- Focus on E-E-A-T (Experience, Expertise, Authoritativeness, Trustworthiness)
- Build backlinks through high-quality, shareable content

### 1.3 Competitive Landscape

**Primary SEO Competitors**:
- Alibaba.com (DA: 93, 500M+ monthly visits)
- GlobalSources.com (DA: 82, 15M+ monthly visits)
- ThomasNet.com (DA: 84, 8M+ monthly visits)
- TradeKey.com (DA: 72, 5M+ monthly visits)

**Our Differentiation**:
- Africa-focused content (underserved niche)
- Cross-border trade expertise (U.S.-Africa compliance)
- Enterprise B2B focus (vs. SMB focus of competitors)

## 8. Technical SEO Requirements

### 8.1 On-Page SEO Checklist

**Every Page/Post Must Have**:
- [ ] Primary keyword in H1 title
- [ ] Primary keyword in URL slug
- [ ] Meta title (50-60 characters, includes primary keyword)
- [ ] Meta description (150-160 characters, compelling CTA)
- [ ] Primary keyword in first 100 words
- [ ] H2/H3 subheadings with secondary keywords
- [ ] Alt text for all images (descriptive, includes keywords where relevant)
- [ ] Internal links to related content (3-5 per post)
- [ ] External links to authoritative sources (1-3 per post)
- [ ] Schema markup (Article, FAQ, HowTo where applicable)
- [ ] Mobile-responsive design
- [ ] Fast page load speed (<3 seconds)

### 8.2 Technical SEO Infrastructure

**Site Architecture**:
```
broxiva.com/
├── /en/ (English)
│   ├── /blog/
│   │   ├── /sourcing/
│   │   ├── /compliance/
│   │   ├── /trade-finance/
│   │   └── /exporting/
│   ├── /guides/ (Pillar pages)
│   ├── /case-studies/
│   ├── /whitepapers/
│   └── /suppliers/[category]/[country]/
├── /fr/ (French)
│   └── (Same structure)
├── /ar/ (Arabic)
│   └── (Same structure)
```

**Core Web Vitals Targets**:
- Largest Contentful Paint (LCP): <2.5 seconds
- First Input Delay (FID): <100 milliseconds
- Cumulative Layout Shift (CLS): <0.1

**Structured Data Implementation**:
- Organization schema (homepage)
- BreadcrumbList schema (all pages)
- Article schema (blog posts)
- FAQPage schema (FAQ sections)
- Product schema (supplier listings)
- AggregateRating schema (supplier reviews)

### 8.3 XML Sitemap Structure

**Separate Sitemaps**:
- sitemap-pages.xml (Static pages)
- sitemap-blog.xml (Blog posts)
- sitemap-guides.xml (Pillar pages)
- sitemap-suppliers.xml (Supplier profiles)
- sitemap-products.xml (Product listings)

**Submission**: Google Search Console, Bing Webmaster Tools

### 8.4 Robots.txt Configuration

```
User-agent: *
Allow: /

# Disallow admin and internal pages
Disallow: /admin/
Disallow: /api/
Disallow: /checkout/
Disallow: /dashboard/

# Sitemap location
Sitemap: https://broxiva.com/sitemap.xml
```
